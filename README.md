# mitman
mitman is a tool to generate README.md and LICENSE.txt of MIT License

### Requirements
* macOS or Ubuntu
* npm

### Usage
**install**
```bash
npm install -g TakutoYoshikai/mitman
```

**generate README.md and LICENSE.txt**
```bash
mitman  <REPO NAME> <AUTHOR NAME>
```
